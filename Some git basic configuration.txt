Find folder directory --> cd folderName

Make the file accesible to githup--> git init

Add the file to gitup---> git add .

Commit the file---> git commit -m "FolderName"

Change githup url to default name---> git remote add defaultName CurrentgitURLLink 

Check if the url name have chnage ---> git remote

Push the file to github repository---> git push -u gitURL
